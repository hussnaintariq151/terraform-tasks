    AWS CLI command to get the list of all Amazon Linux 2 AMIs in the us-east-2 region that match the free-tier-compatible pattern

    aws ec2 describe-images --owners amazon --filters "Name=name,Values=amzn2-ami-hvm-2.0.*-x86_64-gp2" --query "Images[*].[ImageId,Name]" --region us-east-2
